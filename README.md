# ✅ To-Do List Web App (Flask + MySQL)

A full-stack To-Do List web application built using Flask for the backend, MySQL for database management, and HTML/CSS for a clean and responsive frontend.

---

## 📌 Project Overview
This project helps users manage their daily tasks efficiently. Users can add, update, delete, and track their tasks in a simple and user-friendly interface.

---

## 🧠 Features
- Add new tasks
- Update existing tasks
- Delete tasks
- View all tasks
- Data stored in MySQL database
- Responsive UI using HTML & CSS

---

## 🛠️ Tech Stack
- Backend: Python (Flask)
- Database: MySQL
- Frontend: HTML, CSS

## ⚙️ Installation

1. Clone the repository:
git clone https://github.com/muhammad-qasim410/flask.git

2. Navigate to project folder:
cd flask

3. Install dependencies:
pip install -r requirements.txt

4. Setup MySQL database:
- Create a database
- Import database.sql file

5. Configure database in app.py

---

## ▶️ Usage

Run the Flask app:
python app.py

Open in browser:
http://127.0.0.1:5000/
